# 변온컨
## 내용 설명
-교실 온도 제어

-투표로 온도 설정

-비밀번호 입력시 오도 조절가능

## 실행 결과
계절을 입력하시요:여름

인원 수를 입력하시요:1

18/19/20/21/22/23중 하나 선택하시오:19

19도로 설정온도가 설정되었습니다

비밀번호를 입력하세요:11234561

설정온도를 입력하세요:20

설정온도가 20도로 설정되었습니다.

# 지원금 분류
## 내용 설명
-가구의 인원 수와 가구의 소득 수준에 따라 점수 계산 및 지원금액 결정

-결과,기준소득,점수,지원금액,판단사유 출력
## 실행결과
### 입력
현재 국가부채 기준 비율 입력: 60
현재 국가 부채 비율 입력: 45
가구원 수 입력: 4
연소득 입력: 30000000
### 출력<bn>
=== 결과 === <bn>
기준 소득: 6097773
점수: 60
지원금: 700000
판단 사유: 소득 기준 충족
