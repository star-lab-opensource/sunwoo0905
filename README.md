## 반온도 컨트롤러(반온컨)


## Program Algorithm


각 반에서 18~26도에 투표 → 평균 온도 계산 

계산된 평균으로 에어컨/난방 자동 설정(필요 시 비밀번호 입력 후 온도 지정 가능)

## 실행방법


계절과 학생 수를 입력한 뒤, 각 학생이 원하는 온도를 받아 평균을 계산하고 그 값을 최종 온도로 설정한다.

단, 비밀번호를 입력하면 원하는 온도를 직접 지정할 수 있다.
